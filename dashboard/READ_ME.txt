Programı çalıştırmak için adımlar:
1. .Renviron dosyasında ROBYN_API_PROJECT_PATH değerini kendi bilgisayarındaki proje yolunu ver.
2. Projenin olduğu dosyayı working directory olarak ayarla, sağ alttaki penceredeki "More" kısmından yapılabilir.
3. /api/allocator_output/main.R dosyasyını çalıştır ve serveri ayağa kaldır.
4. windows'ta cmd yi açıp projenin içindeki /ui dosyasına git ve npm start komutunu yaz, npm yüklü değilse önce npm install yapılmalı.
